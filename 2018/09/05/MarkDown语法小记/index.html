其他注意事项

  • 在 markdown 中直接使用尖括号<something>会被文本默认为HTML标签语句而不予显示。
    • 使用转义字符&lt;代替<，用&gt;代替>
    • 或者左闭合的尖括号前加一个转义符号\，例如：“\<something>”
